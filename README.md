# IPOcal（股票申購資金試算）

本專案提供一個本機網頁，會自動抓取 TWSE「公開申購公告-抽籤日程表」並試算：

- 每天有哪些可參與的申購
- 參與多檔同時申購時，資金占用期重疊下「需要準備多少錢」
- 已公布的中籤率（TWSE 表內已結案者會有）

## 快速開始（Windows / PowerShell）

安裝依賴：

```bash
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

啟動：

```bash
uvicorn app.main:app --reload
```

打開瀏覽器：

- `http://127.0.0.1:8000/`

## 資金占用假設（MVP）

- **資金凍結起點**：以「申購開始日」當天起算（你實際下單日可能不同，之後可改成以你自己的下單紀錄為準）
- **未中籤退款日**：以「抽籤日次一營業日」估算（先以週末排除，暫不處理國定假日）
- **中籤情境**：資金視為一直占用到「撥券日」

後續可加上：

- 你各券商的實際扣款/退款規則（含假日、手續費/郵寄工本費）
- 多來源整合（上櫃/興櫃、民間整理站、或券商公告）
